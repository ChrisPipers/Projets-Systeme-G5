README 
Utiliser pdflatex pour générer le fichier pdf

Les différents scripts doivent être executés en tant que super utilisateur. Soit en faisant 
su suivis du code super utilisateur ou soit en précédent l'execution du scripts par sudo et entrant 
le code super utilisateur après avoir taper "enter".
La commande pour lancer le script pour changer l'image de fond est :
	sudo ./changeBackground
L'image doit impérativement se trouver dans le répertoire utiliser par ce script, c'est à dire le repertoire
/home/user/Images
	
La commande pour lancer le script pour changer les paramètres GRUB2 est :
	sudo ./scriptSYSG5
Ce script peut-être executer de n'importe dans l'architecture de fichier.
	sudo ./scriptChangeBackground
